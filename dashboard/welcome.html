🚀 Safe Stack - ようこそ

Safe Stack は、Claude Code AI 駆動のフルスタック開発テンプレートです。 Python (FastAPI) バックエンド、TypeScript (React) フロントエンド、marimo ダッシュボードを統合した開発環境を提供します。

🚀 クイックスタート

  1. ターミナルパネル（左下） で以下のコマンドを実行:
    pnpm run dev
    # React + FastAPI のみ
    pnpm run dev:all
    # React + FastAPI + marimo（初回は pnpm run enable:marimo）
  2. 開発サーバーが起動したら、アプリプレビュー（右上・このパネル） のURL入力欄に以下を入力:
フロントエンド /preview/ - React アプリケーション API Docs /api/docs - FastAPI Swagger UI marimo /marimo/ - ダッシュボード

📖 各パネルの使い方

VS Code (左上)
  • コード編集
  • ファイル操作
  • Git管理
アプリプレビュー (右上)
  • URLを入力してプレビュー
  • リロードボタンで再読込
  • 別タブで開くリンクあり
ターミナル (左下)
  • 開発コマンド実行
  • 3つのターミナルを切替可能
  • ZShellで快適な操作
ユーティリティ (右下)
  • メモ: 作業メモを保存
  • ログ: サービスログ確認
  • marimo: 管理ダッシュボード
  • API Docs: FastAPI仕様

🔧 よく使うコマンド

  • pnpm run dev
    フロントエンド + バックエンドを起動
  • pnpm run dev:all
    フロントエンド + バックエンド + marimo を一括起動（初回は pnpm run enable:marimo）
  • pnpm run lint
    コード品質チェック (Python Ruff + TypeScript ESLint)
  • pnpm run format
    コードフォーマット (Python Ruff + TypeScript Prettier)
  • pnpm run test
    テスト実行
  • claude
    Claude Code CLI を起動

💡 ヒント

  • ホットキー: Ctrl+Shift+Alt+D でフルスクリーン切替
  • メモ: ユーティリティパネルのメモタブは自動保存されます
  • ログ: 各サービスのログをリアルタイムで確認できます
  • セキュアモード: ファイアウォールで外部通信を制限しています

🛠️ トラブルシューティング

開発サーバーが起動しない

  • ターミナルで pnpm run dev を実行しているか確認してください。
  • ポート 3000・8000・2718 が他のプロセスで利用されていないか確認してください。
  • pnpm install で依存関係をインストール済みであることを確認してください。

VS Code が表示されない

  • ブラウザのキャッシュをクリアしてみてください
  • 別タブで開くリンクをクリックして確認してください

外部APIに接続できない

  • セキュアモード有効時は ADDITIONAL_ALLOWED_DOMAINS でドメインを許可してください
  • 詳細は docs/environment/security.md を参照
